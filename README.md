# Паяльня PM

SPA для управління проектами і завданнями. Дизайн у стилі Slash — темна тема, obsidian + ember gold.

**[→ Демо](https://payalnya-k9u2iw23h-yaroslav-radchuks-projects.vercel.app/#/)**

---

## Стек

Vue 3 Composition API, TypeScript (без `any`), Pinia, Vite, SCSS + BEM, vue-i18n.

Для форм — vee-validate + zod: схема описується один раз, помилки з'являються після blur і при сабміті. Drag-and-drop через vuedraggable (SortableJS). Графік статистики — Chart.js через vue-chartjs.

HTTP-шар — axios, але бекенду немає: `axios-mock-adapter` перехоплює всі запити ще до мережі і читає/пише дані в `localStorage` із симульованою затримкою. Підключити реальний API — лише змінити `baseURL` у `src/api/client.ts`.

## Архітектура

Логіка таблиць винесена в окремі composables — сортування, фільтрація і ресайз колонок не живуть у компонентах. Кожен composable generic і не залежить від решти.

Підтверджуючі модалки (видалення, архівування) реалізовані через Promise-based composable — виклик виглядає як `await confirm({ ... })`, відповідь повертається після вибору користувача. Це дозволило уникнути локального стану відкриття модалки в кожному компоненті.

Стан фільтрів і режим відображення (table/kanban) зберігаються в `localStorage` між сесіями.

Інтерфейс двомовний — українська і англійська через vue-i18n.

## Що реалізовано по ТЗ

Весь обов'язковий функціонал: таблиця проектів із сортуванням, фільтрацією і ресайзом колонок, сторінка проекту з двома режимами відображення (таблиця + канбан), drag-and-drop усередині колонок і між ними зі збереженням порядку і статусу, форма створення/редагування завдань із валідацією, повний API-шар.

Весь бонусний функціонал: toast-повідомлення після кожної дії, діаграма розподілу завдань на головній сторінці, збереження стану фільтрів між перезавантаженнями, анімації переходів між сторінками, модальними вікнами і картками канбану.

Додатково зверх ТЗ: архівування і відновлення проектів, модалки підтвердження для деструктивних дій, скелетони при завантаженні, юніт-тести (Vitest + Vue Test Utils), локалізація інтерфейсу.
